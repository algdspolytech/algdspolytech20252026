# Лабораторная работа 25 — Задача о рюкзаке (Ruckzack)

## Условие
Даны:
- количество предметов `n`
- вместимость рюкзака `B`
- требуемая суммарная ценность `K`
- массив весов `w[i]`
- массив ценностей `v[i]`

Требуется определить, существует ли набор предметов, суммарная ценность которого не меньше `K`, и при этом суммарный вес не превышает `B`.  
Если решение существует — вернуть индексы выбранных предметов.

## Структура проекта
