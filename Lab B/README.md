# Лабораторная работа — Вариант 23: Поиск кратчайшего пути (матрица смежности)

**Вариант:** 23
**Задача:** граф задан матрицей смежности. Необходимо найти кратчайший по числу рёбер путь между двумя вершинами.
Поддерживаются оба варианта: ориентированный и неориентированный граф (параметр в входном файле).

**Формат входного файла** (пример):
```
n
a11 a12 ... a1n
...
an1 an2 ... ann
s t
d
```
- `n` — число вершин (целое, >0).
- Затем `n` строк по `n` чисел (0/1) — матрица смежности.
- `s t` — индексы вершин (0-based) между которыми нужно найти путь.
- `d` — необязательный параметр: `0` — неориентированный граф (по умолчанию), `1` — ориентированный граф.

Программа выводит два вида результатов: длину кратчайшего пути (число рёбер) и сам путь списком вершин (если путь существует). Если пути нет, выводится сообщение и длина `-1`.
