# Лабораторная работа D вариант 27 — Расстановка ферзей### УсловиеНа шахматной доске NxN расставить N ферзей так, чтобы никакие два не били друг друга.### Входные данныеФайл input.txt — одно целое число N (0 ≤ N < 1 000 000).### Выходные данныеФайл output.txt — N строк:  номер_строки номер_столбца.### Примерinput.txt### Реализация- Алгоритм: backtracking - Функция solve_nqueens(int n, int* out) возвращает расстановку- 10 юнит-тестов### Файлы- nqueens.c, nqueens.h — реализация- NQueensTests.cpp — тесты- README.md, .gitignore
