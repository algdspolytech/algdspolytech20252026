# Лабораторная работа. Вариант 10: Построение списка смежности (1)

## Условие задачи
Граф задан в файле матрицей смежностей. Построить список смежностей.

## Описание
Программа считывает матрицу смежности из текстового файла и преобразует ее в список смежности. Матрица смежности представляет собой квадратную матрицу, где каждый элемент указывает на наличие или отсутствие ребра между вершинами графа.

## Реализованные функции

### Основные функции:
- `read_line` - чтение строки из файла с динамическим выделением памяти
- `graph_list` - преобразование матрицы смежности в список смежности и его вывод
