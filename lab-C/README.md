# Лабораторная работа С - Поиск пути в лабиринте

## Условие задачи
**Вариант 24: Поиск пути в лабиринте (3)**

Вам дан клеточный лабиринт на плоскости. Найдите путь от заданной точки до выхода из лабиринта.

## Описание решения
Программа находит кратчайший путь от стартовой точки до выхода в клеточном лабиринте размером 10×10.

### Обозначения в лабиринте:
- `-1` - стартовая точка
- `-2` - выход из лабиринта  
- `0` - стена (непроходимая клетка)
- `1` - проходимая клетка

### Алгоритм:
1. Поиск координат старта и выхода в матрице лабиринта
2. Рекурсивный обход лабиринта с заполнением расстояний от старта
3. Восстановление кратчайшего пути от выхода к старту
4. Визуализация найденного пути
