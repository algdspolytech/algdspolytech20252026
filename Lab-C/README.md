Описание
Программа для поиска пути в клеточном лабиринте от заданной начальной точки (S) до выхода (E). Реализован алгоритм "правила правой руки", который имитирует движение с приоритетом поворота направо.

Реализованные операции

Проверка корректности ячейки (находится ли она в пределах лабиринта и не является ли стеной '1')
Поиск пути с использованием алгоритма "правила правой руки"
Вывод лабиринта с отмеченным найденным путем (символ '*')

Сборка и запуск
Проект создан в Visual Studio 2022. Для сборки:

Открыть проект в Visual Studio

Нажать Ctrl+Shift+B для сборки

Нажать Ctrl+F5 для запуска программы

При запуске программа предлагает выбрать режим:

1 — Запуск демонстрации на примере лабиринта 5x5.

2 — Запуск всех модульных тестов.

Преимущества и недостатки алгоритма "правила правой руки"
Преимущества:

Простота реализации.

Не требует больших вычислительных ресурсов и памяти (кроме хранения пути).

Гарантированно находит выход в простых связных лабиринтах без петель.

Недостатки:

Может не найти выход в лабиринтах, где выход находится внутри "острова" (т.е. путь требует отрыва руки от стены).

Не гарантирует нахождение кратчайшего пути.

В лабиринтах с петлями может зациклиться (в программе предусмотрен лимит шагов для предотвращения бесконечного цикла).
