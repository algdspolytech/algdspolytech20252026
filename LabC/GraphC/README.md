﻿Лабораторная работа A — Сравнение представлений графа

Цель работы:  
Реализовать программу для чтения и сравнения графов, заданных в виде списка смежности и матрицы смежности. Проверить эквивалентность представлений и корректность работы программы при различных входных данных.

Средства разработки:  
Язык программирования: C  
Среда разработки: Visual Studio  
Тестирование: Google Test

Описание работы:  
Были реализованы структуры Vec и Graph, а также функции для работы с ними. Программа умеет считывать графы из файлов со списком смежности и матрицей смежности, выводить их на экран и проверять, одинаковые ли они. Для проверки использовались три входных файла:  
adjlist.txt, adjmatrix.txt и adjmatrix_not_equal.txt.  
Кроме того, были подготовлены дополнительные файлы для тестирования — badmatrix.txt, badmatrix2.txt и badloop.txt.

Все функции реализованы в статической библиотеке, включающей файлы adj.c и adj.h. Файл main.c используется для демонстрации работы программы и проверки её корректности вручную. В проекте также реализовано освобождение памяти для всех структур, а функции корректно обрабатывают ошибки чтения и некорректные данные.

Тестирование:  
Для проверки корректности работы был создан отдельный проект с модульными тестами на C++ с использованием Google Test. Всего реализовано 10 тестов, которые проверяют чтение корректных и некорректных файлов, сравнение равных и неравных графов, работу с пустыми графами, а также обработку графов с петлями.  
Все тесты успешно пройдены.

Вывод:  
Программа корректно выполняет чтение графов из различных представлений и правильно определяет, совпадают ли они. Реализация позволила закрепить навыки работы с динамическими структурами данных на языке C и познакомиться с основами модульного тестирования с помощью Google Test.
