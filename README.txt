Лабораторная работа D. Вариант 14
Решение головоломки Судоку с помощью алгоритма "Танцующие ссылки" (Dancing Links)
Условие задачи
Решите головоломку судоку размером ( N > 2 ).

Формат входных данных
В текстовом файле input.txt хранится:

Натуральное число ( 2 < N < 10 )
( N^2 ) строк, содержащие ( N^2 ) целых чисел в интервале ( [0; N^2] ) каждая
0 означает, что клетка пуста
Формат выходных данных
В файл output.txt выводится:

Решение судоку в том же формате, что и исходный файл
Если решения не существует - выводится 0
Алгоритм
Задача решается с помощью:

Сведения задачи Судоку к задаче точного покрытия
Алгоритма "Танцующие ссылки" (Dancing Links) Дональда Кнута
Рекурсивного поиска с возвратом (backtracking)
Особенности реализации
Реализована структура данных DLX для точного покрытия
Преобразование судоку в матрицу ограничений 4 типов:
Позиция клетки (row, col)
Число в строке (row, number)
Число в столбце (col, number)
Число в блоке (block, number)
Поиск решения алгоритмом X
