# Лабораторная работа 4.22 — Самый длинный путь

## Условие
Дан неориентированный граф, вершины s и t и целое число K.
Необходимо найти простой путь (без повторения вершин) от s к t
длины не меньше K (по количеству рёбер).

Если путь существует — вывести вершины пути.
Если нет — вывести 0.

## Язык
C (C99)
